Страницы

Для каждой страницы кроме страницы входа в аккаунт требуется авторизация.


Содержимое страницы авторизации

На данной странице располагается форма для авторизации. 
Для входа на сайт надо ввести указанные в регистрации никнейм и пароль.


Содержимое страницы клиентов

На данной странице в виде таблицы отображаются данные о пациентах поликлиники с их контактными данными.


Содержимое страницы услуг

На данной странице в виде таблицы отображается список услуг врачей поликлиники.


Содержимое страницы врачей

На данной странице в виде таблицы отображаются данные о врачах поликлиники с их контактными данными.


Содержимое страницы соотношений врач-услуга

На данной странице в виде таблицы отображаются связи врач-услуга. 
Она показывает врачей с предоставляемой(ыми) услугами, а также их стоимость.


Содержимое страницы записей поликлиники

На данной странице в виде таблицы отображаются записи поликлиники.
